## 环节 1：创建广告系列

### 步骤 1：登录 Google Ads

```
网址：https://ads.google.com
操作：登录您的 Google Ads 账户
```

### 步骤 2：新建广告系列

```
操作路径：
1. 点击左侧菜单"广告系列"
2. 点击蓝色"+"按钮
3. 选择"新广告系列"
```

### 步骤 3：选择广告系列目标

```
页面标题：选择广告系列目标

操作：
1. 选择"网站访问次数"（第一个）
2. 点击"继续"

注意：不要选择"潜在客户"，选择"网站访问次数"更适合私域引流
```

### 步骤 4：选择广告系列类型

```
页面标题：选择广告系列类型

操作：
1. 选择"搜索"（Search）
2. 点击"继续"
```

### 步骤 5：填写基本信息

```

填写内容：
┌─────────────────────────────────────┐
│ 广告系列名称：翡翠定制 - 搜索 - 全球华语区 - 2026Q2 │
│                                    │
│ 起始日期：今天（或选择具体日期）        │
│                                    │
│ 结束日期：留空（不设置）               │
└─────────────────────────────────────┘

操作：
1. 填写广告系列名称
2. 选择起始日期
3. 不设置结束日期
```

### 步骤 3.5：关键词匹配类型说明（重要）

```
三种匹配类型详解：

1. 精确匹配 [关键词]
   示例：[翡翠原石 台湾]
   匹配：翡翠原石 台湾、台湾翡翠原石
   不匹配：翡翠原石定制、缅甸翡翠原石
   特点：最精准，只匹配完全相同或极接近的词
   优势：转化率高，CPA 低
   劣势：展示次数少

2. 词组匹配 "关键词"
   示例："翡翠原石定制"
   匹配：翡翠原石定制、高端翡翠原石定制、翡翠原石定制戒指
   不匹配：定制翡翠原石（词序颠倒）
   特点：必须按顺序包含完整词组
   优势：精准度较高，展示次数适中
   劣势：可能错过一些相关流量

3. 广泛匹配 关键词（无符号）
   示例：翡翠原石定制
   匹配：翡翠原石定制、缅甸翡翠定制、翡翠手镯定制、珠宝定制
   特点：匹配相关搜索，范围最广
   优势：展示次数最多，可能发现新词
   劣势：可能匹配不相关搜索，需配合否定词

使用建议：
- 精确匹配：50% 预算（保证精准流量）
- 词组匹配：35% 预算（平衡精准和覆盖）
- 广泛匹配：15% 预算（测试新词，需配合否定词）
```

## 📞 常见问题处理

### Q1：广告审核不通过怎么办？

```
原因：
- 广告文案包含敏感词
- 落地页内容违规
- 资质证明不足

解决：
1. 查看审核不通过原因
2. 修改广告文案
3. 重新提交审核
4. 如需要，提供资质证明
```

### Q2：广告没有展示怎么办？

```
可能原因：
1. 出价过低
2. 质量得分低
3. 关键词搜索量低
4. 预算已花完

解决：
1. 提高出价
2. 优化广告质量
3. 扩展关键词
4. 增加预算
```

### Q3：CPA 过高怎么办？

```
优化方向：
1. 暂停高成本低转化关键词
2. 优化广告文案相关性
3. 调整受众定位
4. 优化落地页转化率
5. 使用智能出价策略
```
