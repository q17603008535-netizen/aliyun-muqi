### 推荐操作：先跳过，不添加任何受众

**选择原因：**

1. **搜索广告初期**：

## 七、竞价策略

### 推荐策略：Clicks（点击）

**适用阶段：** 新广告系列（0 转化数据）

### 为什么选择 Clicks？

**你的情况：**
- ✅ 新广告系列（0 转化数据）
- ✅ Google 系统不知道你的目标客户是谁
- ✅ 需要快速积累点击和转化数据
- ✅ 预算有限（1000 人民币/天）

**Clicks 策略的优势：**
```
第 1-2 周（数据积累期）：
├── 目标：最大化点击量
├── 好处：快速获得流量
├── 结果：积累 200-400 次点击/天
└── 数据：15-30 次转化/周（达到学习期要求）
```

### CPC 上限设置

#### 台湾地区

```
最高 CPC 上限：12 人民币
原因：
- 台湾 CPC 约 6-12 元
- 避免系统出价过高（可能到 20-30 元）
- 确保预算可以获取足够点击
```

#### 美国地区

```
最高 CPC 上限：18 人民币
原因：
- 美国 CPC 约 10-18 元
- 避免系统出价过低（抢不到流量）
- 确保广告可以展示
```

### 为什么不设置更高？

**不设置上限的风险：**
```
❌ 系统可能出价 30-50 元/次点击
❌ 1000 元预算只能买 20-30 次点击
❌ 数据积累太慢
❌ 浪费预算
```

**设置上限的好处：**
```
✅ 控制成本在合理范围
✅ 1000 元可以买 80-150 次点击
✅ 快速积累数据
✅ 测试不同关键词效果
```
