## 一、核心功能缺失（影响业务）

### 1. ❌ 表单提交功能
- **优先级**：🔴 P0 - 紧急
- **位置**：[src/components/Contact.astro](src/components/Contact.astro#L128-L137)
- **当前状态**：
  ```javascript
  form.addEventListener('submit', (e) => {
    e.preventDefault();
    alert('感謝您的咨詢！我們將盡快與您聯繫。');
    form.reset();
  });
  ```
- **问题描述**：
  - 表单仅使用 `alert()` 提示
  - 无实际数据提交
  - 无邮件通知
  - 无成功/失败反馈
- **影响**：
  - 用户无法通过网站发送咨询
  - 需要手动添加微信/LINE
  - 影响业务转化率
- **技术方案**：
  - **推荐**：Formspree（免费额度足够，易集成）
  - **备选**：EmailJS（功能强大，免费额度有限）
- **预计工时**：1 天
- **实现步骤**：
  1. 注册 Formspree 获取 Form ID
  2. 修改表单 `action` 属性
  3. 添加成功/失败反馈 UI
  4. 添加表单验证
  5. 测试邮件通知
