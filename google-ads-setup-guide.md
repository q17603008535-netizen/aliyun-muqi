## 附录 B：常见问题解答

### Q1：为什么我的广告没有展示？
```
可能原因：
1. 出价过低
2. 质量得分太低
3. 关键词搜索量太低
4. 受众定位太窄
5. 预算已花完

解决方案：
1. 提高出价
2. 优化广告质量
3. 扩展关键词
4. 放宽受众
5. 增加预算
```

### Q2：CPA 过高怎么办？
```
优化方向：
1. 优化落地页转化率
2. 暂停高成本低转化关键词
3. 调整受众定位
4. 优化广告文案相关性
5. 使用智能出价策略
```

### Q3：如何提高质量得分？
```
三个要素优化：
1. 预期点击率：优化广告文案
2. 广告相关性：关键词与广告匹配
3. 落地页体验：优化加载速度、内容相关性
```

### Q4：预算应该何时增加？
```
增加预算的时机：
1. CPA 低于目标且转化稳定
2. ROI 持续为正且>100%
3. 广告评级持续前 3
4. 有明显季节性需求增长

增加幅度：
- 每次增加 20-30%
- 观察 3-5 天后再调整
- 避免大幅波动
```
