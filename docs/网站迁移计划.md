# 沐祁珠宝品牌改版开发规范文档

## 🎯 改版目标

**核心目的**：为了符合 Google Ads 和 Facebook Ads 广告投放审核要求，将网站从"洛天翡翠"品牌改版为"沐祁珠宝"，并将服务定位从"代购"升级为"定制化服务"，以提升网站在广告平台的信任度和审核通过率。

### Google/FB 广告审核关键要求
1. ✅ **独立品牌形象**：品牌名称简洁、专业，避免关键词堆砌
2. ✅ **高端服务定位**：从"代购"（中介商）升级为"定制专家"（专业服务商）
3. ✅ **品牌一致性**：全站统一品牌名称，避免混乱
4. ✅ **企业正规性**：展现 15 年经验，体现资深企业形象

### 改版策略
- **品牌名称**：洛天翡翠 → 沐祁珠宝（Muqi Jewelry）
- **服务定位**：翡翠原石代购 → 缅甸翡翠定制专家
- **经验展示**：曼德勒专业买手 → 15 年匠心传承
- **业务模式**：代购 → 定制
