## 📊 广告系列架构总览

### 整体结构

```
广告系列（1 个）
└── 广告组（8 个）
    ├── 地区定向广告组（6 个）：台湾、香港、澳门、新加坡、美国、加拿大
    └── 通用广告组（2 个）：珠宝定制服务、高端翡翠收藏
```

### 架构设计逻辑

**采用"单系列多广告组"策略的原因**：

1. ✅ **数据集中积累**：所有广告组共享一个广告系列的预算和数据，有利于 Google AI 快速学习
2. ✅ **预算灵活调配**：可根据各地区表现实时调整预算分配
3. ✅ **管理效率高**：单一广告系列便于统一管理和优化
4. ✅ **质量得分提升**：数据集中有助于快速提升账户整体质量得分
5. ✅ **智能出价优化**：足够的转化数据支持智能出价策略

**预算分配策略**：

| 广告组 | 预算占比 | 每日预算 | 优先级 |

| 翡翠原石 - 香港 | 15% | 150 CNY | 中 |

| 高端翡翠收藏 - 通用 | 动态分配 | - | - |

#### 优化期（第 8-30 天）- 表现导向

**假设台湾、新加坡、美国表现最佳**：

| 广告组 | 预算占比 | 每日预算 | 调整依据 |
|--------|---------|---------|---------|
| 翡翠原石 - 台湾 | 30% | 300 CNY | CPA < 60, CTR > 4% |
| 翡翠原石 - 新加坡 | 30% | 300 CNY | CPA < 65, CTR > 3.5% |
| 翡翠原石 - 美国 | 30% | 300 CNY | CPA < 70, CTR > 3.5% |
| 其他地区 | 10% | 100 CNY | 维持测试 |

**预算调整规则**：

```
IF CPA < 目标 CPA AND 转化数 >= 5 THEN
  增加预算 20-30%
ELSE IF CPA > 目标 CPA * 1.5 THEN
  减少预算 30-50%
ELSE IF CPA > 目标 CPA * 2 OR 转化数 = 0 THEN
  暂停广告组
END IF
```

### 3. 出价策略设置

**三阶段出价策略**：

#### 阶段 1：数据积累期（第 1-7 天）

```
策略：最大化点击（Maximize Clicks）
目标：快速积累点击和转化数据
预期 CPC：3-8 CNY
预期点击：125-333 次/天
预期转化：15-20 个
```

**设置方法**：
```
1. 选择"出价" > "更改出价策略"
2. 选择"最大化点击"
3. 设置"每次点击费用上限"：8 CNY（防止过高 CPC）
4. 保存
```

#### 阶段 2：优化期（第 8-21 天）

```
策略：目标转化成本（Target CPA）
目标 CPA：65 CNY（可接受范围 50-80 CNY）
预期转化：12-20 个/天
预期 CPA：55-75 CNY
```

**切换条件**：
- ✅ 过去 7 天转化数 >= 15 个
- ✅ 转化追踪正常运作
- ✅ 数据波动趋于稳定

**设置方法**：
```
1. 选择"出价" > "更改出价策略"
2. 选择"目标转化成本"
3. 设置目标 CPA：65 CNY
4. 保存并观察 3-5 天
```

#### 阶段 3：放量期（第 22-30 天）

```
策略：目标转化成本（Target CPA）或 目标广告支出回报率（Target ROAS）
目标 CPA：60-100 CNY（如 ROI 良好可适当提高）
或
目标 ROAS：200%（每投入 1 元回报 2 元）
预期转化：15-25 个/天
```

**切换条件**：
- ✅ 过去 14 天转化数 >= 30 个
- ✅ CPA 稳定在目标范围内
- ✅ ROI > 150%

**设置方法**：
```
1. 评估 ROI 数据
2. 如 ROI 稳定 > 200%，可选择 tROAS
3. 设置目标 ROAS：200%
4. 或继续优化 tCPA
```

### 4. 投放网络设置

**推荐配置**：

```
✅ 搜索网络：包括 Google 搜索结果页面
❌ 搜索合作伙伴网站：初期不启用（流量质量不可控）
❌ Google 展示广告网络：单独创建展示广告系列
```

**设置原因**：

1. ✅ **搜索网络**：用户主动搜索，意向明确，转化率高
2. ❌ **搜索合作伙伴**：流量质量参差不齐，初期难以控制 CPA
3. ❌ **展示网络**：用户被动接收，适合品牌曝光而非转化

**后续扩展**：
- 测试期结束后，可单独创建"展示广告系列"做再营销
- 可创建"PMax 广告系列"覆盖全渠道

### 7. URL 跟踪参数设置

**配置参数**：

```
跟踪模板：{lpurl}?utm_source=google&utm_medium=cpc&utm_campaign={campaignid}&utm_content={adgroupid}&utm_term={keyword}

最终到达网址扩展：不选中
```

**UTM 参数说明**：

| 参数 | 值 | 说明 |
|------|-----|------|
| utm_source | google | 流量来源 |
| utm_medium | cpc | 广告媒介 |
| utm_campaign | {campaignid} | 广告系列 ID（自动替换） |
| utm_content | {adgroupid} | 广告组 ID（自动替换） |
| utm_term | {keyword} | 触发关键词（自动替换） |

**示例 URL**：
```
原始 URL：https://yourdomain.com/contact

追踪 URL：
https://yourdomain.com/contact?utm_source=google&utm_medium=cpc&utm_campaign=123456789&utm_content=987654321&utm_term=翡翠原石定制
```

**设置方法**：
```
1. 进入"设置" > "广告系列 URL 选项"
2. 填写"跟踪模板"
3. 不选中"最终到达网址扩展"
4. 保存
```

## 🔧 广告组创建步骤

### 批量创建方法（推荐）

**使用 Google Ads 编辑器**：

1. **下载并安装 Google Ads 编辑器**
   ```
   下载地址：https://adseditor.google.com/
   ```

2. **下载账户数据**
   ```
   打开编辑器 > 获取账户 > 选择账户 > 下载
   ```

3. **批量添加广告组**
   ```
   广告系列 > 右键 > 添加广告组
   或使用 Excel 导入功能
   ```

4. **批量添加关键词**
   ```
   选择广告组 > 关键字和定位 > 添加关键字
   或从 CSV 文件导入
   ```

5. **批量添加否定关键词**
   ```
   广告组级别否定关键字列表
   或创建共享否定关键字列表
   ```

6. **上传更改**
   ```
   查看更改 > 上传 > 确认
   ```

### 手动创建方法

**步骤 1：创建广告组**
```
1. 进入广告系列
2. 点击"广告组和关键字"
3. 点击蓝色"+"按钮
4. 输入广告组名称
5. 设置出价（如需要）
6. 保存
```

**步骤 2：添加关键词**
```
1. 选择广告组
2. 点击"关键字和定位"
3. 点击蓝色"+"按钮
4. 输入关键词（每行一个）
5. 选择匹配类型
6. 保存
```

**步骤 3：添加否定关键词**
```
1. 选择广告组
2. 点击"否定关键字"
3. 点击蓝色"+"按钮
4. 输入否定关键词
5. 保存
```

**步骤 4：设置受众定位**
```
1. 选择广告组
2. 点击"受众群体"
3. 点击蓝色"+"按钮
4. 选择"浏览"
5. 添加人口统计、兴趣等
6. 设置出价调整
7. 保存
```
